# 作品流程展示

Work Process View 是一个简洁直观的作品制作流程展示组件，适用于插画、3D建模等多阶段视觉对比。

它支持通过淡入淡出（Fade）方式，展示线稿、建模、渲染等不同阶段的图像与对应文字，方便观众清晰了解作品的创作过程。

## 项目特色
- 多阶段图像对比切换
- 拖动滑块实时查看不同阶段的图片
- 支持自定义色彩

## 怎么用呢？

1. 将 work-process-view 相关 HTML、CSS、JS 文件引入项目
2. 按模板结构放置图片与文字区域
3. 自定义配置文字、图像路径及样式变量

## 用在哪好呢？
- 作品、插画集
- 3D 建模流程演示
- 教学案例展示
- 视觉特效对比
- ...
